# React.js - Syllabus

## Course Overview

This course covers modern React.js development including component architecture, state management, routing, authentication, and advanced patterns. Students build full-featured single-page applications.

## Type

Online / Asynchronous

## Prerequisites

- JavaScript fundamentals (ES6+)
- Basic HTML/CSS

## Course Schedule

| Week    | Topic                             | Description |
|---------|-----------------------------------|-------------|
| Week 1  | React Basics                      | JSX, components, props, state, hooks |
| Week 2  | React Router                      | Client-side routing, nested routes, navigation |
| Week 3  | Role-Based Access Control (RBAC)  | Route guards, user roles, conditional rendering |
| Week 4  | Logging and Monitoring            | Error boundaries, logging libraries, performance tracking |
| Week 5  | Lazy Loading                      | Code splitting, React.lazy, Suspense |
| Week 6  | Redux                             | Global state management with Redux Toolkit |
| Week 7  | Backend Authentication            | JWT, session management, auth flows |
| Week 8  | Authentication Part 1             | Login/signup forms, token storage |
| Week 9  | Authentication Part 2             | Refresh tokens, protected routes |
| Week 10 | Authentication Part 3             | OAuth, social login integration |
| Week 11 | Authentication Part 4             | Multi-factor authentication patterns |
| Week 12 | Concurrent React                  | useTransition, useDeferredValue, concurrent features |
| Week 13 | Server Components                 | React Server Components, streaming SSR |
| Week 14 | SWR and React Query               | Data fetching, caching, and mutation patterns |
