# Rust Programming - Syllabus

## Course Overview

This course covers the Rust systems programming language and its applications in modern software development, including blockchain and Solana smart contract development.

## Type

Online / Asynchronous

## Prerequisites

- Experience with at least one systems language (C, C++, or Java)
- Basic understanding of memory management concepts

## Course Schedule

| Week   | Topic                          | Description |
|--------|--------------------------------|-------------|
| Week 1 | Rust Fundamentals              | Ownership, borrowing, lifetimes, type system, cargo |
| Week 2 | Rust Programming Deep Dive     | Structs, enums, pattern matching, error handling, traits |
| Week 3 | Rust Concurrency               | Threads, channels, Arc/Mutex, async/await |
| Week 4 | Solana and Blockchain Basics   | Blockchain fundamentals, Solana architecture, accounts model |
| Week 5 | Solana Smart Contracts (Rust)  | Writing and deploying programs using Anchor framework |
| Week 6 | Final Project                  | Build and deploy a Rust application or Solana program |

## Grading
