# Microsoft Excel Charts

## Why Use Charts?

Charts help turn raw numbers into visual information. Instead of reading many rows of data, a chart lets you quickly identify trends, comparisons, and proportions.

![Common Excel chart types](/blog/images/spreadsheet/excel_chart_types.svg)

## When to Use Each Chart Type

| Chart Type | Best for |
|---|---|
| Column chart | Comparing values across categories |
| Bar chart | Comparing many category labels horizontally |
| Line chart | Showing change over time |
| Pie chart | Showing parts of a whole |
| Scatter plot | Showing relationship between two numeric variables |

## Sample Data

| Month | Sales |

## Create a Column Chart

1. Select the data range, including headings.
2. Click **Insert**.
3. Choose **Column Chart**.
4. Add a clear chart title such as `Monthly Sales`.

This chart works well when you want to compare January, February, March, and April side by side.

## Create a Line Chart

Use a line chart when the data represents time.

Example:

- monthly sales
- weekly attendance
- daily temperature

The line chart makes trends easier to see.

## Chart Formatting Tips

- Use meaningful titles
- Label axes clearly
- Avoid too many colors
- Keep the chart simple
- Remove unnecessary 3D effects

## Example: Comparing Departments

| Department | Budget |
|---|---:|
| HR | 15000 |
| IT | 28000 |
| Sales | 22000 |
| Marketing | 18000 |

Best chart: **bar chart** or **column chart**

## Pie Chart Example

| Category | Percent |
|---|---:|
| Rent | 40 |
| Food | 25 |
| Transport | 15 |
| Savings | 20 |

Best chart: **pie chart**

Use a pie chart only when:

- the total is meaningful
- categories are limited
- values represent parts of one whole

## Chart Elements

Important chart elements include:

- chart title
- axis titles
- legend
- data labels
- gridlines

## Common Mistakes

- using a pie chart for too many categories
- forgetting to include column headings
- using unreadable colors
- comparing unrelated data in the same chart
