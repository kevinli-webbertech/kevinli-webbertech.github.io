# Spring Boot - Syllabus

## Course Overview

This course covers full-stack Java application development with Spring Boot, from basic REST APIs to database access, security, testing, and Kubernetes deployment.

## Type

Online / Asynchronous

## Prerequisites

- Java programming (OOP concepts)
- Basic SQL knowledge
- Maven or Gradle familiarity

## Course Schedule

| Week   | Topic                               | Description |
|--------|-------------------------------------|-------------|
| Week 1 | Spring Boot Introduction            | Project setup, auto-configuration, starters, REST API basics |
| Week 2 | Data Access                         | Spring Data JPA, repositories, CRUD operations |
| Week 3 | Kubernetes Deployment               | Containerizing Spring Boot apps, deploying to K8s |
| Week 4 | MVC Web Development                 | Thymeleaf templates, controllers, model binding |
| Week 5 | JPA Advanced                        | Relationships, JPQL, lazy vs eager loading |
| Week 6 | Lombok                              | Reducing boilerplate with @Data, @Builder, @Slf4j |
| Week 7 | Testing                             | Unit tests, @SpringBootTest, MockMvc, Testcontainers |
| Week 8 | Security                            | Spring Security, JWT auth, role-based access control |
