# AI / Machine Learning - Syllabus

## Course Overview

This course introduces students to practical AI application development using modern large language models (LLMs) and AI APIs. Students will learn how to build and deploy lightweight AI-powered applications.

## Type

Online / Asynchronous

## Prerequisites

- Basic Python programming
- Familiarity with command-line tools
- A valid API key for an LLM provider (OpenAI, Qwen, etc.)

## Tools & Frameworks

- Python 3.9+
- Streamlit
- OpenAI Python SDK (openai-compatible)
- python-dotenv

## Course Schedule

| Week   | Topic                              | Description |
|--------|------------------------------------|-------------|
| Week 1 | Introduction to LLMs and AI APIs  | Overview of large language models, API concepts, and OpenAI-compatible endpoints |
| Week 2 | Environment Setup                 | Virtual environments, `.env` configuration, API key management |
| Week 3 | Qwen Small App Example            | Build a streaming chat app with Streamlit and Qwen model |
| Week 4 | Prompt Engineering Basics         | System prompts, temperature, max tokens, and message history |
| Week 5 | Deploying AI Apps                 | Local deployment, Streamlit Cloud, Docker basics |
