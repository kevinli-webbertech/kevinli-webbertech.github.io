# Linux Administration - Syllabus

## Course Overview

This course provides comprehensive coverage of Linux system administration, from basic command-line usage through shell programming, networking, security, and system debugging tools.

## Type

Online / Asynchronous

## Prerequisites

- Basic computer literacy
- Access to a Linux machine or VM (Ubuntu recommended)

## Course Schedule

| Week    | Topic                         | Description |
|---------|-------------------------------|-------------|
| Week 1  | Linux Distributions           | Overview of major distros, choosing a distro |
| Week 2  | Window Managers               | Desktop environments, tiling WMs, headless servers |
| Week 3  | Linux Commands                | Essential commands: ls, cp, mv, chmod, grep, find |
| Week 4  | Filesystem Hierarchy          | FHS, mount points, inodes, disk usage |
| Week 5  | User Account Management       | useradd, passwd, groups, sudo, PAM |
| Week 6  | Profiles and Environment      | .bashrc, .profile, PATH, environment variables |
| Week 7  | Permissions                   | rwx, octal notation, ACLs, setuid/setgid |
| Week 8  | Devices and Storage           | block devices, fdisk, lsblk, LVM |
| Week 9  | Storage Configuration         | Mount, fstab, NFS, RAID overview |
| Week 10 | /etc Directory                | Key config files: fstab, hosts, passwd, shadow |
| Week 11 | SSH                           | Key-based auth, ssh config, port forwarding |
| Week 12 | Packages and Software         | apt, yum/dnf, snap, building from source |
| Week 13 | Shell Scripting Intro         | bash basics, variables, conditionals |
| Week 14 | Shell Programming             | Functions, loops, arrays, scripts |
| Week 15 | Advanced Scripting            | Process substitution, traps, cron jobs |
| Week 16 | Networking                    | ip, netstat, ss, tcpdump, routing |
| Week 17 | Firewall                      | iptables, ufw, firewalld |
| Week 18 | GCC and GDB                   | Compiling C programs, debugging with gdb |
| Week 19 | ld and Linker                 | Static vs dynamic linking, ld options |
| Week 20 | Debugging Utilities           | strace, ltrace, lsof, top, htop |
| Week 21 | SELinux                       | Mandatory access control, policies, contexts |
| Week 22 | GPU Administration            | NVIDIA drivers, CUDA setup, GPU monitoring |
| Week 23 | Ubuntu Tools                  | Ubuntu-specific admin utilities |
| Week 24 | Editors: Vim and Nano         | Efficient editing in terminal-based editors |
