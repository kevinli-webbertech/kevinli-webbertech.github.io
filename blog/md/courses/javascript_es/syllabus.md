# JavaScript (ES6+) - Syllabus

## Course Overview

This course covers modern JavaScript (ES6 and beyond), including language fundamentals, DOM manipulation, asynchronous programming, and web APIs. Students build interactive web applications from scratch.

## Type

Online / Asynchronous

## Prerequisites

- Basic HTML and CSS knowledge
- Familiarity with any programming language is helpful

## Course Schedule

| Week    | Topic                         | Description |
|---------|-------------------------------|-------------|
| Week 1  | JavaScript Introduction       | History, role in the web, environment setup |
| Week 2  | JavaScript Features           | ES6+ features: let/const, arrow functions, destructuring, spread |
| Week 3  | Enablement and Placement      | Script loading, module system, browser vs Node.js |
| Week 4  | JavaScript Basics             | Variables, types, operators, expressions |
| Week 5  | Lab: GitHub Copilot + JS      | Using AI assistance for JavaScript development |
| Week 6  | DOM and Events                | Document object model, event listeners, bubbling |
| Week 7  | Control Flow                  | if/else, switch, loops, iterators |
| Week 8  | Functions                     | Declarations, expressions, closures, higher-order functions |
| Week 9  | Classes and OOP               | ES6 classes, inheritance, prototypes |
| Week 10 | Browser Object Model (BOM)    | window, navigator, location, history |
| Week 11 | Asynchronous JavaScript       | Callbacks, Promises, async/await |
| Week 12 | Web APIs                      | Fetch API, REST calls, JSON handling |
| Week 13 | Final Project                 | Build and present a complete web app |
