# Machine Learning - Syllabus

## Course Overview

This course introduces the theory and practice of machine learning, covering classical algorithms, neural networks, and modern frameworks. Students implement and evaluate models on real datasets.

## Type

Online / Asynchronous

## Prerequisites

- Python programming
- Basic statistics and linear algebra
- NumPy and Pandas familiarity

## Course Schedule

| Week   | Topic                                     | Description |
|--------|-------------------------------------------|-------------|
| Week 1 | Machine Learning Introduction             | Types of ML, supervised/unsupervised/reinforcement, workflow |
| Week 2 | Mathematics for Neural Networks           | Linear algebra review, calculus basics, gradient descent |
| Week 3 | Neural Networks                           | Perceptrons, activation functions, backpropagation |
| Week 4 | Classic ML Algorithms                     | Linear regression, logistic regression, decision trees, k-NN |
| Week 5 | Scikit-Learn Introduction                 | Pipeline, estimators, datasets, cross-validation |
| Week 6 | Hyperparameters and Model Validation      | Grid search, k-fold CV, bias-variance tradeoff |
| Week 7 | Final Project                             | Apply ML to a real dataset; present results |
