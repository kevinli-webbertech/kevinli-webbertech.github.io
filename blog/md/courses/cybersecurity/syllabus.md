# Cybersecurity - Syllabus

## Course Overview

This course covers modern cryptographic algorithms, security protocols, and practical security tools used in real-world systems. Students learn both the mathematical foundations and applied usage of cryptography.

## Type

Online / Asynchronous

## Prerequisites

- Basic programming (Python or Java)
- Basic understanding of networking (TCP/IP)

## Course Schedule

| Week    | Topic                              | Description |
|---------|------------------------------------|-------------|
| Week 1  | Public and Private Keys            | Asymmetric cryptography fundamentals |
| Week 2  | Encryption Algorithms Overview     | Symmetric vs asymmetric, use cases |
| Week 3  | RSA                                | Key generation, encryption, decryption |
| Week 4  | ECC, EIGamal, ECDH                 | Elliptic curve cryptography and key exchange |
| Week 5  | AES                                | Block cipher, modes of operation |
| Week 6  | Hashing: SHA-1, SHA-2, SHA-3       | Hash functions and collision resistance |
| Week 7  | SHA-256, MD5                       | Practical hashing applications |
| Week 8  | Password Hashing: argon2, bcrypt   | Secure password storage |
| Week 9  | scrypt, blake3                     | Modern cryptographic hash functions |
| Week 10 | Security Application Patterns      | Combining algorithms in real systems |
| Week 11 | OpenSSH and SSH Authentication     | Key-based SSH, config hardening |
| Week 12 | SSL/TLS, X.509, PEM                | Certificate chains, HTTPS internals |
| Week 13 | GPG, JWT                          | Email security and web token standards |
| Week 14 | Certificate Authorities & Tools    | PKI, security tooling overview |
