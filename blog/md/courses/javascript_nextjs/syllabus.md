# Next.js Framework - Syllabus

## Course Overview

This course introduces the Next.js full-stack React framework, covering routing, server-side rendering, and modern UI features. Students build and deploy production-ready web applications.

## Type

Online / Asynchronous

## Prerequisites

- JavaScript fundamentals (ES6+)
- React basics (components, props, state)

## Course Schedule

| Week   | Topic                         | Description |
|--------|-------------------------------|-------------|
| Week 1 | Next.js Basics                | Project setup, file-based routing, pages vs app directory, SSR vs CSR |
| Week 2 | Drag and Drop UI              | Building interactive drag-and-drop interfaces in Next.js |
| Week 3 | API Routes                    | Creating backend API endpoints within a Next.js app |
| Week 4 | Data Fetching                 | getServerSideProps, getStaticProps, SWR, React Query |
| Week 5 | Deployment                    | Vercel deployment, environment variables, build optimization |
