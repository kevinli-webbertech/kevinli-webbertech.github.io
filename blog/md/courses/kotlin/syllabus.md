# Kotlin Programming - Syllabus

## Course Overview

This course covers the Kotlin programming language from fundamentals to advanced features, with a focus on Android-compatible and server-side development including Spring Boot integration.

## Type

Online / Asynchronous

## Prerequisites

- Basic programming experience (Java or Python recommended)

## Course Schedule

| Week    | Topic                         | Description |
|---------|-------------------------------|-------------|
| Week 1  | Kotlin Basics                 | Syntax, REPL, tooling, comparison with Java |
| Week 2  | Variables                     | val vs var, type inference, nullable types |
| Week 3  | Data Types                    | Primitive types, strings, type casting |
| Week 4  | Program Control               | if/when/for/while, ranges, loops |
| Week 5  | Scope Functions               | let, run, apply, also, with |
| Week 6  | Functions and Classes         | Function types, default args, extension functions |
| Week 7  | Class Constructors            | Primary and secondary constructors, init blocks |
| Week 8  | Data Classes                  | Immutable models, copy, equals/hashCode, toString |
| Week 9  | Singleton (Object)            | Object declarations, companion objects |
| Week 10 | Collections                   | List, Set, Map, sequences, lambdas |
| Week 11 | Exception Handling            | try/catch, sealed classes for errors |
| Week 12 | Functional Programming        | Higher-order functions, map/filter/reduce |
| Week 13 | Delegation                    | by keyword, observable properties |
| Week 14 | Productivity Features         | Coroutines intro, null safety, idiomatic Kotlin |
| Week 15 | Kotlin + Spring Boot          | Building a REST API with Spring Boot in Kotlin |
